    Cenários

    É uma estratégia reconhecida para compreender as interações entre ambientes e sistemas, assim como elicitar a parte comportamental do software, sua dinâmica e/ou seu fluxo.

    Cada cenário geralmente cobre um pequeno número de interações possíveis, e oferecem diversos tipos de informação em diferentes níveis de detalhamento.

    C1

    Realizar cadastro

    Objetivo
    Permitir que os usuários se cadastrem
    Contexto
    Local: Tela inicial,clicando no segundo botão para se cadastrar
    Pré-Condição: Ter instalado o app
    Pós-Condição: realiza inscrição
    Atores
    Usuário que deseja ter cadastro no aplicativo
    Recursos
    Internet, aplicativo, conta de e-mail
    Episódios
    Usuário deseja reportar seu estado de saúde
    Usuário navega até a página Home
    Clica em algum dos botões "BEM" ou "MAL"
    Se a opção selecionada for bem, o aplicativo salva o estado de saúde
    Se caso for MAL, pede mais informações e em seguida, salva as informações
    Restrição
    possuir um e-mail
    Exceção
    Estar offline
    E-mail inválido
    Aplicativo estar indisponível
    Senha inválida
    Conta já existente

    C2

    Fazer login

    Objetivo
    Fazer Login no aplicativo
    Contexto
    Local: Tela inicial deve clicar no botão login
    Pré-Condição: Ter o app instalado
    Pós-Condição: Usuário faz login
    Atores
    Usuário que deseja fazer login e já fez inscrição
    Recursos
    Internet, aplicativo, e-mail válido, senha válida
    Episódios
    Usuário clica no botão "login" na tela inicial. Em seguida, inserir os dados de login
    estará logado
    Restrição
    Usuário ter o aplicativo
    Usuário possuir cadastro
    Exceção
    Internet cair
    Usuário esquecer e-mail
    Usuário esquecer a senha

    C3

    Reportar estado de saúde

    Objetivo
    Reporta estado de saúde
    Contexto
    Local: Tela Home deve clicar no botão "BEM" ou MAL"
    Pré-Condição: Ter o app instalado
    Pós-Condição: Usuário registra o estado de saúde
    Atores
    Usuário que quer reportar o seu estado de saúde
    Recursos
    Internet, aplicativo
    Episódios
    Usuário quer reportar o seu estado de saúde abre o aplicativo

    Usuário clica no botão "BEM" é referente ao seu estado de saúde normal.
    Usuário clica no botão "MAL" na tela home. Em seguida, responde mais perguntas sobre os sintomas.
    O sistema salva a opção reportarda.
    Restrição
    Usuário ter o aplicativo instalado.
    Usuário estar logado
    Exceção
    Internet cair
    Usuário ja ter reportado naquele dia
    Usuário não logado
    Sistema indisponível

    C4

    Visualizar mapa

    Objetivo
    Vizualizar mapa
    Contexto
    Local: Aba mapa no menu inferior
    Pré-condição: Estar logado no app
    Pós-condição: Mapa visualizado
    Atores
    Usuário que deseja saber como está a situação de risco de contágio na região que se encontra
    Recursos
    Internet
    Aplicativo
    Dispositivo móvel com GPS
    Episódios
    Usuário abre o aplicativo
    Usuário seleciona mapa menu inferior
    Usuário analisa a região
    Usuário decide quanto cuidado tomar baseado nas marcações do mapa
    Restrição
    Aplicativo estar instalado
    Usuário logado
    Exceção
    Internet não funcionar
    Servidor indisponível

    C5

    Visualizar calendário

    Objetivo
    Visualizar calendário e estatísticas dos relatos
    Contexto
    Local: Aba diário no menu inferior
    Pré-condição: Usuário querer saber como está seu histórico de relatos de estado de saúde
    Pós-condição: Histórico visualizado
    Atores
    Atores
    Usuário que deseja vizualizar o histórico e dados dos relatos
    Recursos
    Aplicativo
    Internet
    Episódios
    Usuário entra no aplicativo
    Usuário seleciona diário no menu inferior
    Usuário olha quantas vezes ele reportou seus status como bem e mal
    Usuário navega pelo calendário e vê quais dias ele reportou e quais não
    Usuário desliza o calendário para o lado e vê as estatísticas dos status
    Restrição
    Aplicativo instalado
    Usuário logado
    Exceção
    Internet não funcionar
    Servidor indisponível

    C6

    Editar dados do perfil

    Objetivo
    Editar dados de algum perfil associado à conta
    Contexto
    Menu flutuante na home, opção Editar perfis
    Pré-condição: Usuário quer modificar informações de algum dos perfis associado à conta
    Pós-condição: Infomações modificadas
    Atores
    Usuário que deseja atualizar os dados de cadastro no aplicativo
    Recursos
    Aplicativo
    Internet
    Episódios
    Usuário entra no aplicativo
    Usuário abre o menu esquerdo da página inicial
    Usuário seleciona Editar perfis no menu
    Usuário seleciona o perfil que deseja editar
    Usuário faz as modificações e salva
    Restrição
    Aplicativo instalado
    Usuário logado
    Exceção
    Internet não funcionar
    Servidor indisponível

    C7

    Visualizar notícias

    Objetivo
    Visualizar noticias relacionadas ao contexto da Saúde
    Contexto
    Local:Tela principal na seção “Notícias”
    Pré-condição: Acesso a internet , Usuário cadastrado, app instalado
    Pós-condição: Tela aberta referente a notícias
    Atores
    Usuários do Guardiões da saúde
    Recursos
    Conta no Guardiões da Saúde
    Usuário busca por notícias antigas
    Acesso a um smartphone
    Episódios
    Usuário clica na opção de notícias na tela principal
    Usuário busca por notícias antigas
    Usuário clica nas noticias sendo redirecionado na pagina do Twitter referente aos guardiões
    Restrição
    Aplicativo instalado
    Usuário logado
    Exceção
    Internet não funcionar
    Usuário não cadastrado
    Email invalido

    C8

    Visualizar Dicas

    Objetivo
    Visualizar Dicas relacionadas ao contexto da Saúde
    Contexto
    Local:Tela principal na seção “Dicas”
    Pré-condicão: Acesso a internet , Usuário cadastrado, app instalado
    Pós-condição: Tela aberta referente a Dicas
    Atores
    Usuários do Guardiões da saúde
    Recursos
    Conta no Guardiões da Saúde
    Usuário busca por dicas relacionadas a saúde
    Acesso a um smartphone
    Episódios
    Usuário tem a dúvida sobre farmácias, hospitais na região e busca a aba Dicas para informações.
    O usuário tem qualquer dúvida relacionada a Covid-19 como uso de máscaras, prevenção, suspeita de infecção, etc, e busca a seção Dicas do app.
    Usuário busca por apoio psicológico durante a pandemia e procura ajuda na seção Dicas.
    Usuário clica nas noticias sendo redirecionado na pagina do Twitter referente aos guardiões.
    Restrição
    Aplicativo instalado
    Usuário logado
    Exceção
    Internet não funcionar
    Usuário não cadastrado
    Email invalido
